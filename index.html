Step 1: Login to Uber

Login to Uber


  1. Login to Uber
  2. Enter your mobile number
  3. Text YOUR_PHONE_NUMBER from the number you entered...
    • For a list of Uber products available at the start address: Products/START ADDRESS
    • To request an Uber product from a start address to an end address: PRODUCT NAME/START ADDRESS/END ADDRESS
  4. You will receive a text with, based on the text you sent, either a list of Uber products or confirmation of your Uber request. If you requested an Uber product, you will receive a text when your request has been accepted.

To make this suitable for production, you would need to disable the use of the sandbox and no longer use the "/v1/sandbox/requests/{request_id}" endpoint to manually modify the status of a request. For now, most functionality is operational and the hack is suitable for demo purposes. Requesting an Uber product currently results in receiving updates via text with the sandbox's demo data for the "processing" and "accepted" request statuses.
